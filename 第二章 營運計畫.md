## 2-1 可行性分析
| 項目 |                                  | 可行性 |
| ---- |:------------------------------  |:------:|
| 市場 | 市場有一定需求                     |   是   |
|      | 具有潛在市場                      |   是   |

|      | 是否有正確的技術知識來完成專案目標    |   是   |
